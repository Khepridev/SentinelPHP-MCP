Usage & Prompts

Maximize the potential of SentinelPHP with these optimized prompts.

Basic Analysis

The simplest command to scan a specific file.

analyze SecureDB.php for SQL injection

Advanced Pattern Learning

Instruct the server to use its Knowledge Base for smarter detection.

1. Show me SQL injection patterns from the 'prompts/' folder
2. Learning from those patterns, analyze 'User.php'
3. Check specifically for FIND_IN_SET vulnerabilities
4. Generate a Proof-of-Concept exploit if found

Protocol-Based Audit

Perform a rigorous audit following the PROTOCOL.md rules.

Perform a full security audit on 'login.php' following the PROTOCOL.md rules. Focus on:
- Zero Trust (validate all inputs)
- SQL Injection in WHERE clauses
- XSS in error messages
- DoS in loops

Knowledge Base Commands

Interact with the server's long-term memory.

Show me the knowledge base statistics
What vulnerability patterns have you learned so far?
